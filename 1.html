HTML

Hypertext Markup Language (HTML) is the standard language for
creating web pages and web applications.
Web browsers receive HTML documents from a web server
or from local storage and render them into multimedia web pages.
HTML describes the structure of a web page semantically
and originally included cues for the appearance of document
It can be assisted by technologies such as Cascading Style Sheets (CSS)
and scripting languages such as JavaScript.
